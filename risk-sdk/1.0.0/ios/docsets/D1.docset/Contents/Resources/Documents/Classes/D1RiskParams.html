    Parameters

    view

    The current view getting the events. On most cases, this is ViewController ‘s view.
    placementName

    The placement name to identify the flow associated with the current page. Supported values (so far) are:

    • “CreateAccountMobile”, for user registration page
    • “LoginMobile”, for login page
    • “TransferFundsMobile”, for transfer funds page
    placementPage

    The page number within the current flow. For single page flow, the value should be 1. For flow that spans multiple page, the value is the page number (1,2,3,…).

The class to provide initialisation function to initialise \mbox{\hyperlink{protocol_secure_log-p}{Secure\+Log}} config, it takes the parameters from \mbox{\hyperlink{interface_secure_log_config_components}{Secure\+Log\+Config\+Components}}. All of parameters is read-\/only and can\textquotesingle{}t be changed after the init\+With\+Config\+Components\+Builder function is called

CommonUiDelegate

@objc
public protocol CommonUiDelegate

UI Delegate that contains methods for managing authenticator selection and optional authenticator additional description.

Since

1.0.0
  • idCloudClient(_:authenticators:userChoiceHandler:cancelHandler:)

    Called during authenticator selection. The developer should show authenticators to the user and let user choose which to use. This method will not be called if one of following condition is fulfilled:

    • There is only one authenticator option in the request.
    • There are two authenticators option; but system biometric is not enrolled. In this case, PIN authenticator will be selected.
    • authenticator - Authenticator selected from the list.

  • idCloudClient(_:authenticatorDescription:authenticatorDescriptionHandler:cancelHandler:)

    Called during enrollment. The developer could potentially provide a UI to the user to allow the user to set additional information regarding a newly enrolled authenticator.

    • authenticatorDescriptor - An object containing custom information on the authenticator.

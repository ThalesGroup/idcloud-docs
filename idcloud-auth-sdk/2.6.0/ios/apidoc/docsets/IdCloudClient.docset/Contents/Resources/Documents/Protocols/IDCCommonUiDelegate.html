Swift

protocol IDCCommonUiDelegate

UI Delegate that contains methods for managing authenticator selection and optional authenticator additional description.

@since 1.0.0

  • -idCloudClient:authenticators:userChoiceHandler:cancelHandler:

    Called during authenticator selection.

    The developer should show authenticators to the user and let user choose which to use.

    This method will not be called if one of following condition is fulfilled:

    • There is only one authenticator option in the request.

    • There are two authenticators option; but system biometric is not enrolled. In this case, PIN authenticator will be selected.

    @since 1.0.0

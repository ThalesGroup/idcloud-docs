Detailed Description

Class to set different configuration parameters for initialization of ThreatMetrix Signal group Pass this object to initialize() for enabling signal collection under ThreatMetrix signal group. This class contains wrapper method for all config APIs of ThreatMetrix SDK. Internally all methods make a call to corresponding config methods in ThreatMetrix SDK.

Since
2.0.0

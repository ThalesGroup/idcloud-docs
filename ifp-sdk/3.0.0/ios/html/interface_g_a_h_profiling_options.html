Detailed Description

Class to set different configuration parameters for profiling/signal collection of ThreatMetrix Signal group. This class contains wrapper method for Profiling option APIs of ThreatMetrix SDK. Internally all methods make a call to corresponding methods in ThreatMetrix SDK.

Since
2.0.0

◆ disableBehavioSec

- (void) disableBehavioSec

Method to disable TMXBehavioSec . If not set, TMXBehavioSec will automatically run during profiling. If set, the TMXBehavioSec module is disabled.

\doxysubsection{Detailed Description}
Class to set different configuration parameters for initialization of Threat\+Metrix Signal group Pass this object to initialize() for enabling signal collection under Threat\+Metrix signal group. This class contains wrapper method for all config APIs of Threat\+Metrix SDK. Internally all methods make a call to corresponding config methods in Threat\+Metrix SDK. \begin{DoxySince}{Since}

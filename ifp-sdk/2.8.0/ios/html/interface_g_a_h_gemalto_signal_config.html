Detailed Description

Class to set different configuration parameters for initialization of all basic signal groups. This config object has to be passed as a second param to initialization() By default all params in this class will have a default value set, if not passed.

◆ signalCacheTimeout

- (double) signalCacheTimeout
readwritenonatomicassign

@double signalCacheTimeout @abstract In seconds, maximum time for which prefetched signals will be cached. Signal values older than this timeout will not be considered during risk calculation, GAH SDK will collect a fresh value during very next call to requestVisitId() after Signal cache timeout. This timeout will override all previous configurations for the cache time out such as SessionIDTimeout in GTMTMXConfig. @discussion Valid at init time to set signalCacheTimeout Default value is 5 Minutes(set value in seconds ex: 300 seconds)
